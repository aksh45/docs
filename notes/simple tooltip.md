# Simple tooltip


### Example
```html
<button class='ui teal button' id='reparse_portal_data' data-tooltip='Convert uploaded raw "portal data json" into "Transformed portal data".'>
```

### Preferences
```
data-tooltip='content of the popup'
data-position='top left'
```
